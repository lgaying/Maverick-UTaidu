字符串的一些方法

January 03 2020 #字符串

描述

  • 修改字符串的大小写
  • 合并（拼接）字符串
  • 添加和删除空白
  • 使用制表符或换行符来添加空白
  • 删除空白

方法

修改字符串首字母为大写

  • string.title()

注：以首字母大写的方式显示每个单词

例子

>>> print ("hello".title())

Hello

将字符串改为全部大写

  • string.upper()

    例子

>>> print ("hello".upper())

HELLO

将字符串改为全部小写

  • string.lower()

    例子

>>> print ("HEllo".lower())

hello

合并（拼接）字符串

  • +

    例子

>>> first_name = "ada"
>>> last_name = "lovelace"
>>> full_name = first_name + " " + last_name

>>> print(full_name)

ada lovelace

使用制表符或换行符来添加空白

  • \n , \t

    例子

要在字符串中添加制表符，可使用字符组合\t

>>> print("Python")
Python

>>> print("\tPython")
    Python

要在字符串中添加换行符，可使用字符组合\n ：

>>> print("Languages:\nPython\nC\nJavaScript")

Languages:
Python
C
JavaScript

删除空白

  • rstrip, lstrip, strip

    例子

>>> favorite_language = ' python '

>>> favorite_language.rstrip() #删除末尾的空白

' python'

>>> favorite_language.lstrip() #删除开头的空白

'python '

>>> favorite_language.strip()  #删除两端的空白

'python'
